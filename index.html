Largest Game: Wizard's Terror

Game Trailer

This was my first medium sized game in Unity, took about 2 months of 2hr/day work. All Art was extra assets, I only did the Programming. It has Platformer Movement, Melee Combat and Ranged Combat with multiple types of Arrows, a variety of Enemies, and progression through an Upgrade System which keeps its data saved when you close the game.

Game Engine: Cookie

Gameplay Demo

Cookie Engine is a game engine that I worked on for about 3.5 months, with an average of 2hr/day work. It is made using C++, uses OpenGL for Rendering, PhysX for Physics. It uses ECS (Entity Component System Architecture) for creating games in it, and for it's own code itself.
